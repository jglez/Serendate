# Serendate Frontend

UI-only mobile app prototype for planning fun date ideas with variability in budget and time while factoring in weather and local events.

## Stack

- Expo + React Native (single codebase for iOS and Android)
- TypeScript

## Run

1. Install dependencies: `npm install`
2. Start Expo: `npm run start`
3. Open in iOS simulator, Android emulator, or Expo Go.

## Scope

- Frontend-only, no backend integration
- Mock local data for venues, weather, and limited-time events
- Discover, Plan, and Saved tabs
- Interactive controls for time, budget, vibe, neighborhood, and weather-smart mode
